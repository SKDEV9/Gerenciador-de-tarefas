# Gerenciador-de-tarefas
Esse eh um programa simples que tem como objetivo gerenciar tarefas passadas pelo usuario. Basicamente o usuario deve inserir uma tarefa na qual ele desjesa realizar. O programa oferece algumas funcionalidades basicas, como: Adicionar uma tarefa; visualizar as suas tarefas; marcar a tarefa como concluida; Excluir a tarefa.
